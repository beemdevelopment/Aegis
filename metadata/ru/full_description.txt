Аутентификатор Aegis это бесплатное, безопасное и свободное приложение для управления вашими токенами двухфакторной аутентификации (2FA) для ваших онлайн-сервисов.

<b>Совместимость</b>
Aegis поддерживает алгоритмы HOTP и TOTP. Эти два алгоритма являются стандартными и широко поддерживаемыми в отрасли, что делает AEGIS совместимой с тысячами сервисов. Любая вебсайт поддерживающий Google Authenticator также будет работать с Aegis.

<b>Шифрование и биометрическая разблокировка</b>
Все ваши одноразовые пароли хранятся в хранилище. Если вы решите установить пароль (что настоятельно рекомендуется), хранилище будет зашифровано с помощью сильной криптографии. Если кто-то с плохим намерением получает файл хранилища, то он не сможет извлечь содержимое не зная пароля. Ввод вашего пароля каждый раз когда вам нужен доступ к одноразовому паролю может быть неудобным. К счастью, вы также можете включить биометрическую разблокировку, если у вашего устройства есть датчик биометрии (то есть отпечаток пальца или разблокировка по лицу).

<b>Организация</b>
Со временем вы, вероятно, накопите десятки записей в своем хранилище. Aegis имеет множество вариантов организации для облегчения поиска токенов. Установите пользовательский значок для записи, чтобы облегчить поиск. Поиск по имени учетной записи или имени сервиса. Есть много одноразовых паролей? Добавьте их в пользовательские группы для легкого доступа. Личное, Работа и Социальные сети - каждому можно назначить свою группу.

<b>Резервные копии</b>
Чтобы убедиться что вы не потеряете доступ к вашим онлайн учетным записям, Aegis может делать автоматическое резервное копирование хранилища куда вы пожелаете. Если ваш облачный провайдер поддерживает Storage Access Framework (SAF) Android (как например NextCloud), он может даже создать автоматические резервные копии в облаке. Создание ручного экспорта хранилища также поддерживается.

<b>Безболезненный переход</b>
Чтобы облегчить переход Aegis может импортировать записи многих других аутентификаторов, в том числе: 2FAS Authenticator, Authenticator Plus, Authy, AndOTP, FreeOTP, FreeOTP+, Google Authenticator, Microsoft Authenticator, Steam, Authenticator и Winauth (потребуется root доступ для приложений которые не умеют экспортировать).
